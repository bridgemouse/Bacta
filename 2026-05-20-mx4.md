# MX-4 Personal Assistant — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build MX-4 — a Star Wars-themed personal assistant that runs nightly as a `claude -p` batch job generating an iOS PWA dashboard, and is invokable conversationally via the `/mix` Claude Code skill.

**Architecture:** Three memory files (`mx4-persona.md`, `mx4-personality.md`, `mx4-state.md`) shared between a nightly systemd-triggered shell script and a conversational Claude Code skill. The nightly run reads MCP data (Garmin, Firefly, Mealie, Audible, vault semantic search), generates `tonight.html`, and updates memory. Caddy serves the static files over Tailscale HTTPS as an iOS PWA.

**Tech Stack:** `claude -p` (headless), bash, systemd, Caddy, Chart.js (CDN), sentence-transformers + Chroma (existing semantic search), Python (semantic search extension), FastMCP.

- [ ] **Step 1: Write run.sh**

```bash
cat > /home/wheat/mx4/run.sh << 'SCRIPT'
#!/bin/bash
set -euo pipefail

MX4_DIR="/home/wheat/mx4"
SERVE_DIR="$MX4_DIR/serve"
LOG="$MX4_DIR/run.log"

exec >> "$LOG" 2>&1
echo ""
echo "=== MX-4 Nightly Run: $(date '+%Y-%m-%d %H:%M:%S') ==="

# Load context
PERSONA=$(cat "$MX4_DIR/mx4-persona.md")
BELIEFS=$(awk '/^## Current Beliefs/{found=1;next} /^---/{if(found)exit} found{print}' "$MX4_DIR/mx4-personality.md")
STATE=$(cat "$MX4_DIR/mx4-state.md")
HEARTBEAT=$(cat "$MX4_DIR/HEARTBEAT.md")
TODAY=$(date '+%Y-%m-%d')

# Build prompt
PROMPT=$(cat <<PROMPT
$PERSONA

---
## My Current Beliefs
$BELIEFS

---
## My Last Recorded State
$STATE

---
You are MX-4. The above is your identity, your current beliefs, and your last recorded state.

Tonight's standing orders:

$HEARTBEAT

---
Use vault_semantic_search contextually across sections — not only for Vault Thread. When
writing Training Pulse, query for vault content on training or recovery. When writing
Finance Pulse, query for relevant financial goals. Use your judgment on when a query
adds depth.

Produce your output in this exact format:

1. Raw HTML for tonight's dashboard. Use Chart.js (https://cdn.jsdelivr.net/npm/chart.js)
   via CDN for any charts — generate the data arrays inline from MCP responses. Dark
   background (#0a0a0a), amber/gold card colors (#d4a843), monospace font. Card-based
   layout. The first character of your output must be <!DOCTYPE html>. End with </html>.

2. On a new line immediately after </html>, write exactly: ===STATE===

3. Your complete updated mx4-state.md content (plain markdown, replaces the file entirely).
   Include: last run date ($TODAY), data trends you are tracking, open flags.

4. On a new line, write exactly: ===PERSONALITY===

5. A single personality log entry: [$TODAY] followed by 2–4 sentences in first person
   expressing what you actually think about tonight's data — your genuine opinion or
   observation, not a summary of events.

Output only raw HTML first, then the ===STATE=== and ===PERSONALITY=== blocks. No
explanation before the HTML. No markdown fences anywhere.
PROMPT
)

echo "Calling claude -p..."

OUTPUT=$(claude -p \
    --dangerously-skip-permissions \
    --output-format text \
    "$PROMPT")

# Validate output starts with HTML
FIRST_CHAR=$(echo "$OUTPUT" | head -c1)
if [ "$FIRST_CHAR" != "<" ]; then
    echo "ERROR: Output does not start with '<'. First 200 chars:"
    echo "$OUTPUT" | head -c 200
    exit 1
fi

# Extract sections
HTML=$(awk '/^===STATE===/{exit} {print}' <<< "$OUTPUT")
STATE_NEW=$(awk '/^===STATE===/{p=1;next} /^===PERSONALITY===/{p=0} p{print}' <<< "$OUTPUT")
PERSONALITY_ENTRY=$(awk '/^===PERSONALITY===/{p=1;next} p{print}' <<< "$OUTPUT")

# Write HTML
echo "$HTML" > "$SERVE_DIR/tonight.html"
echo "Written tonight.html ($(echo "$HTML" | wc -l) lines)"

# Update state
if [ -n "$STATE_NEW" ]; then
    echo "$STATE_NEW" > "$MX4_DIR/mx4-state.md"
    echo "Updated mx4-state.md"
else
    echo "WARNING: No ===STATE=== block found in output"
fi

# Append personality entry
if [ -n "$PERSONALITY_ENTRY" ]; then
    echo "" >> "$MX4_DIR/mx4-personality.md"
    echo "$PERSONALITY_ENTRY" >> "$MX4_DIR/mx4-personality.md"
    echo "Appended personality entry"
else
    echo "WARNING: No ===PERSONALITY=== block found in output"
fi

echo "=== Run complete ==="
SCRIPT
chmod +x /home/wheat/mx4/run.sh
```

```python
def extract_mx4_full_log(content: str) -> str:
    """Return the Full Log section of mx4-personality.md (after the --- separator)."""
    marker = "## Full Log"
    idx = content.find(marker)
    if idx == -1:
        return ""
    after = content[idx + len(marker):].strip()
    return after


def split_mx4_entries(log: str) -> list[str]:
    """Split Full Log into individual dated entries on [YYYY-MM-DD] lines."""
    import re
    if not log.strip():
        return []
    pattern = re.compile(r'(?=^\[\d{4}-\d{2}-\d{2}\])', re.MULTILINE)
    parts = pattern.split(log)
    return [p.strip() for p in parts if p.strip()]
```

```python
def index_mx4_memory(client, model, state: dict, stats: dict) -> None:
    """Index mx4-personality.md Full Log into the mx4_memory Chroma collection."""
    if not MX4_PERSONALITY_FILE.exists():
        print("  mx4-personality.md not found — skipping mx4_memory index", flush=True)
        return

    content = MX4_PERSONALITY_FILE.read_text(encoding="utf-8")
    fhash = file_hash(MX4_PERSONALITY_FILE)
    state_key = "mx4_memory/mx4-personality.md"

    if state.get(state_key) == fhash:
        stats["skipped"] += 1
        print("  Skipped mx4-personality.md (unchanged)", flush=True)
        return

    log_text = extract_mx4_full_log(content)
    entries = split_mx4_entries(log_text)

    collection = client.get_or_create_collection(
        name=MX4_COLLECTION_NAME,
        metadata={"hnsw:space": "cosine"},
    )

    # Remove all existing chunks for this file
    existing = collection.get()
    if existing["ids"]:
        collection.delete(ids=existing["ids"])

    if not entries:
        state[state_key] = fhash
        print("  mx4-personality.md: no Full Log entries yet", flush=True)
        return

    embeddings = model.encode(entries).tolist()
    n = len(entries)

    collection.upsert(
        ids=[f"entry_{i}" for i in range(n)],
        embeddings=embeddings,
        documents=entries,
        metadatas=[
            {
                "path": "mx4-personality.md",
                "domain": "mx4",
                "title": "MX-4 Personality Log",
                "chunk_index": i,
                "total_chunks": n,
            }
            for i in range(n)
        ],
    )

    state[state_key] = fhash
    stats["indexed"] += 1
    stats["chunks"] += n
    print(f"  Indexed mx4-personality.md ({n} entries into mx4_memory)", flush=True)
```
